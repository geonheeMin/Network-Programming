**1) 물리계층 (Physical Layer)**

데이터(0과 1의 나열) 아날로그 신호로 바꾸어 전선으로 흘려보내고
(encoding)

아날로그 신호가 들어오면 데이터로 해석하여
(decoding)

물리적으로 연결된 두 대의 컴퓨터가 0과 1의 나열을 주고받을 수 있게 해주는 모듈.
